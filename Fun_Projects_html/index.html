telltalebot

telltalebot is a twitterbot completely developed in Shiny. telltalebot describes some brilliant pictures from The Commons, a public photography archives from Flickr. The Shiny application telltalebot is automatically scheduled to post tweets multiple times in a day. Please don't forget to follow us for more interesting tweets!

Image Content Analysis

Image Content Analysis is an R Shiny Application powered by Microsoft Cognitive Services Computer Vision API. It returns information about visual content found in an image. The algorithms use tagging, descriptions and domain-specific models to identify content and label it with confidence.
